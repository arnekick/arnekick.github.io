Stage

Angular developer bij E.care

Tijdens het laaste jaar van mijn studies in de Toegepaste Informatica van Thomas More liep ik gedurende 13 weken stage bij het bedrijf E.care. E.care is een bedrijf dat software ontwikkeld, verkoopt en implementeerd die het medisch process in ziekenhuizen ondersteunt. Hun software concentreet zich vooral op de afdleingen Spoedgevallen en Operatiekwartier. E.care vind het nodig om hun software webbased te maken zodat hun software performanter wordt en er frisser gaat uitzien.

Mijn stageopdracht bestond uit het omzetting van bestaande modules van de E.care software naar hun web based variant in Angular voor zowel computer als tablet. Voor elk van deze bestaande modules moest er een webbased variant worden gerealiseerd. Tijdens de stage hebben we 5 modules in angular gerealiseerd. telkens als we aan een nieuwe module begonnen moest er eerst een prototype worden gemaakt hieruit werden dan de beste aspecten samengevoegd. Zo onstond dan de applicatie die we telkens gingen ontwerpen.

Tijdens mijn stage hebben we gemerkt dat sommige stukken code in meerdere applicaties gebruikt worden. Ik heb dan de opdracht gekregen om uit te zoeken hoe ik een library opbouw in angular en deze kan publiceren. Hierbij heb ik veel geleerd want dit hadden zowel de developpers van E.care en mezelf nog nooit gedaan. Nadat ik de library had gemaak theb ik documentatie moeten schrijven over alle how to's die hierbij kwamen kijken en de errors die ik heb gehad tijdens het opbouwen van de library. Op deze manier weten zij ook hoe ze in de toekomst de library nog kunnen uitbreiden zonder dit allemaal opnieuw uit te zoeken.

Ik ben tijdens mijn stage nog een paar andere frameworks/libraries moeten gaan onderzoeken om te kijken of deze voldeden aan de eisen die nodig waren. Zo heb ik de EO.WebBrowser van EssentialObject onderzocht en de Web documenteditor voor angular van TxTextControl.

Onderaan vind je de bijhorende documenten van mijn stage inclusief de handleiding voor het opbouwen van libraries.
